# Solve Me First FP

[hackerrank](https://www.hackerrank.com/challenges/fp-solve-me-first)

An A+B problem.

### Input Format

`You are given two integers, A and B on separate lines.`

### Output Format

`Output an integer that denotes A+B`

### Constraints

`1 <= A, B <= 1,000`

### Sample Input

```
2
3
```

### Sample Output

```
5
```
