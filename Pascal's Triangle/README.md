# Pascal's Triangle

[hackerrank](https://www.hackerrank.com/challenges/pascals-triangle)

### Input Format  

`A single line of input, integer K.`  

### Constraints
`1 <= K <= 10`

### Output Format  

`Output the first K rows of Pascal's triangle.`  

### Sample Input  

```
4
```  
### Sample Output  

```
1
1 1
1 2 1
1 3 3 1
```
